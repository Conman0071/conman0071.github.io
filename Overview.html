About Mekanism:

Mekanism is an independant mod that implements early, mid and Late game technology and machines into Minecraft.
The mod ustilises the three added ores, "Osmium", "Copper" and "Tin" as well as most of the base game resources.
As well as working by itself it can also be used to shortcut other major mods such as IC2 and Modular Power Suits.

Download:

Mekanism needs forge modloader to run on base game minecraft.
To download Mekanism mod click here. There are 3 parts to the
mod, the core which contains all of the core aspects of the mod,
MekanismTools which adds the tools to the mod and Mekanism
Generators which adds generators to the mod. The core is required
but the other two are optional and rely on the core module to work.
